◆概要
私が、習慣化したいことに関する情報にすぐアクセスできるようにするサポートwebアプリです。

トレーニングであれば、各部位のトレーニング動画が再生される機能。
瞑想であれば瞑想する時間に応じて適した動画が再生される機能。
ヨガであれば、ヨガする時間に応じて適した動画が再生される機能。
フロントエンドであれば、新規開発や基礎学習、ポートフォリオ作成などを促す機能。
読書であれば、今読んでいるタイトルの追加、更新、削除、検索、完了（チェックボックスオン）などが行える機能。
ブログであれば、執筆やカスタマイズ、テーマ考案などを促す機能。

私は使用するために作成しました。

◆デザイン
シンプルかつわかりやすいを意識したデザインになっています。

◆実装した機能について
トレーニング、瞑想、ヨガに関しては、基本的に動画を再生するだけの機能を実装しています。
フロントエンド、ブログに関しては、アプリ上で実行できるというよりは、行動を促すような仕組みにしています。
読書に関しては、To-Doリストの機能（追加、削除、検索、更新）を実装しています。

◆使用言語
HTML,CSS,JavaScript,JQuery,PHP
